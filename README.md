## 🚀 Quick Start

```bash
# Install dependencies
./00-INSTALL/setup.sh

# Start reconnaissance
python3 10-RECON/resonance_scanner.py --target <authorized_target>

# Run vulnerability scan
python3 20-SCANNING/network_scanner.py

# Check system integrity
python3 50-REPORTING/integrity_checker.py
```
